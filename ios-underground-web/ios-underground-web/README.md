# iOS Underground - Web Version

This is the production-ready web version of the iOS Underground Telegram Mini App.

## Quick Deploy

### Option 1: Vercel (Recommended)
```bash
npm i -g vercel
vercel
```

### Option 2: Netlify
```bash
npm i -g netlify-cli
netlify deploy --prod --dir .
```

### Option 3: Any Static Host
Simply upload all files in this folder to your hosting provider.

### Option 4: Docker
```dockerfile
FROM nginx:alpine
COPY . /usr/share/nginx/html
EXPOSE 80
```

## Files Included
- `index.html` - Main application
- `assets/` - CSS and JavaScript bundles
- `__manus__/` - Analytics and tracking
